üìÑ What is Content Sharing?

Content sharing allows BrowseEasy's AI to read and understand the current webpage content. When enabled, you can:

  • Ask questions about the page content
  • Get summaries of the main information
  • Find specific information on the page
  • Get accessibility suggestions based on the actual content

üîß How to Test

Step 1: Enable Content Sharing

  1. Open the BrowseEasy extension panel
  2. Look for the checkbox "üìÑ Share current page content with AI"
  3. Check the box to enable content sharing
  4. You should see a status like "Ready: X words"

Step 2: Try These Questions

  • "What is this page about?"
  • "Summarize the main content"
  • "What links are available on this page?"
  • "What images are on this page?"
  • "What forms are available?"
  • "What are the main headings?"
  • "How can I make this page more accessible?"
