BrowseEasy Hover Highlighting Test

Test Instructions:

  1. Open the BrowseEasy extension panel
  2. Important: If "Highlight Links" is enabled, you'll see yellow highlighting on links permanently
  3. Enable "Highlight on Hover" in the Settings tab
  4. Hover over each element below to test the functionality
  5. Elements marked in GREEN should show blue hover highlighting
  6. Elements marked in RED should NOT show any hover highlighting
  7. Key difference: Hover highlighting is blue with dashed outline, permanent link highlighting is yellow with solid border
Visual Guide:
• Blue dashed outline = Hover highlighting (only on hover)
• Yellow solid border = Permanent link highlighting

Test Results

After testing, the hover highlighting should:

  • Only highlight truly interactive elements when you hover over them
  • Show a blue background with dashed blue outline (different from yellow link highlighting)
  • Include a subtle animation/transition
  • Not interfere with the element's functionality
  • Disappear immediately when the cursor moves away
  • Override permanent link highlighting while hovering (blue replaces yellow temporarily)
Troubleshooting:
• If you see elements highlighted when NOT hovering, that's likely the "Highlight Links" feature
• Hover highlighting should ONLY appear when your cursor is over an element
• The hover effect should be blue/dashed, not yellow/solid
• Try disabling "Highlight Links" to see only the hover effect
